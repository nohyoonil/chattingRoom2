# CHATINGROOM2 

실시간 채팅 서비스를 베이스로 하여 구현해보고 싶던 기술들(__소셜로그인, 실시간채팅, CI/CD__)을 접목해본 프로젝트입니다.


# 기술 스택
java / springboot / redis / spring security / h2 / docker / github actions
